Instrucciones para el Programador

  • Verificar requisitos: Asegúrese de que el servidor cumpla con los requisitos de SamarioPHP.
  • Base de datos: Cree una base de datos vacía antes de iniciar la instalación.
  • Archivos: Verifique que todos los archivos del proyecto estén correctamente ubicados en el servidor.
  • Ejecutar la instalación: Haga clic en "Iniciar instalación" para comenzar el proceso de instalación de tablas y configuraciones.
  • Seguridad: Una vez completada la instalación, elimine el archivo de instalación o configúrelo para que no sea accesible desde la web.

Instrucciones de Instalación

Sigue los pasos a continuación para completar la instalación inicial del framework:

Paso 1: Configurar el archivo de configuración

Edita el archivo samariophp/configuracion.php para configurar las credenciales de la base de datos y los ajustes de la aplicación:


return [
    'base_de_datos' => [
        'tipo' => 'mysql',
        'servidor' => 'localhost', // Cambia este valor por el servidor de tu base de datos
        'puerto' => 3306, // Cambia el puerto si es necesario
        'nombre_basedatos' => 'nombre_base_datos', // Nombre de tu base de datos
        'nombre_usuario' => 'usuario', // Usuario de la base de datos
        'clave_usuario' => 'contraseña', // Contraseña de la base de datos
        'conjunto_caracteres' => 'utf8', // Ajusta la codificación según tus necesidades
    ],
    'aplicacion' => [
        'nombre' => 'Nombre de tu aplicación', // Nombre de tu aplicación
        'url_base' => 'http://localhost/', // Cambia a la URL base de tu aplicación
        'mantenimiento' => false, // Cambia a true para habilitar el modo de mantenimiento
        'entorno' => 'desarrollo', // Cambia a 'produccion' cuando lances la aplicación
    ],
];

Paso 2: Ejecutar el instalador

Accede al archivo de instalación desde tu navegador o desde la línea de comandos:

  • Navegador: Ve a {{ config.aplicacion.url_base }}instalacion.
  • CLI: Ejecuta el comando php samariophp/instalador.php desde la raíz del proyecto.
Nota: El instalador verificará si las tablas ya existen. Si están creadas, no se realizarán cambios adicionales.

Paso 3: Confirmar la instalación

Una vez completada, deberías ver el mensaje "Instalación completada con éxito." en el navegador o terminal.

Detalles adicionales:

  • Durante la instalación, se crearán las siguientes tablas en la base de datos:
    • usuarios (para gestionar la autenticación y roles)
    • roles (para gestionar los permisos)
    • usuarios_roles (para asociar usuarios con roles)
    • sesiones (para la gestión de sesiones activas de usuarios)
    • tokens_api (para la gestión de accesos mediante API)
  • Si necesitas modificar las tablas, puedes hacerlo directamente a través del archivo esquema_inicial.php y regenerar las migraciones con el instalador.

¡Eso es todo! El framework está listo para usar.
